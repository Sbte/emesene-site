--- name: msn-protocol-resources layout: post title: MSN protocol resources time: 2009-01-13 20:14:00 +01:00 --- if you want to learn how the msn protocol works, here I give you some links to read :)

  • Hypothetic: pretty outdated but useful to start..
  • MsnPiki: a wiki with updated content
  • MsnFanatic forum: a forum you know :P, you can find interesting threads there
  • Siebe documentation: great resource for p2p, I don't know why he removed it (using internet archive)
  • imfreedom: new but pretty empty
  • some useful msn msdn links :D
The tools I use to debug and find new features are wireshark and ospy, both great tools, ospy is easier to use, but you need windows and WLM

Orfeo C recommended SocketSniff, I did't tested it, but it may be useful for someone
